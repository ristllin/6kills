# Hello Skill

This is a template skill that shows the expected layout for skills in this repo.

## What a skill contains

1. **Frontmatter** (`name` + `description`) — Claude reads the `description` to decide when to apply the skill, so make it specific about triggers.
2. **Body** — clear, actionable instructions for the task.
3. **Optional sibling files** — scripts, templates, or reference docs in the same folder, referenced from here so they load only when needed.

## How to use this template

1. Copy this folder to `skills/<your-skill-name>/`.
2. Update the `name` and `description` in the frontmatter.
3. Replace this body with your instructions.
4. Delete this example skill when you no longer need it.
